## Installation

**npm:**

```sh
npm install @davikiyo/simple-ui-components
```

**yarn:**

```sh
yarn add @davikiyo/simple-ui-components
```

## Components

The available components are as follows:

- AppBar
- Card
- Drawer
- Grid
- Icon
- Modal
- Pagination
- Table
- UI
  - Button
  - Checkbox
  - IconButton
  - Textbox
